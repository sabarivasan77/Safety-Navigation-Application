# SafeRoute - Quick Start Guide

## 🚀 Getting Started

### First Launch
1. Open the application in your browser
2. You'll see a welcome overlay - read it and click "Get Started"
3. The map will center on Coimbatore (default location)

## 📍 Finding Your Route

### Method 1: Text Search
1. Click on "Start Location" search bar
2. Type your location (e.g., "RS Puram, Coimbatore")
3. Select from auto-suggestions
4. Repeat for "Destination"
5. Routes will automatically calculate

### Method 2: Map Selection
1. Click "Select on map" button under Start Location
2. Click anywhere on the map to set start point
3. Click "Select on map" for destination
4. Click on map to set end point
5. Routes will automatically calculate

## 🗺️ Understanding the Map

### Color Zones
- **Green Circles** - Safe areas (70%+ safety score)
- **Yellow Circles** - Moderate safety (50-70%)
- **Red Circles** - Risk areas (<50%)

### Map Markers
- **Blue Markers** 🔵 - Police Stations
- **Red Markers** 🔴 - Hospitals
- **Green Markers** 🟢 - Petrol Bunks

### Route Colors
- **Blue (Thick)** - Selected route
- **Gray (Thin)** - Alternative routes

## 📊 Reading Safety Scores

### Overall Score
- Displayed as percentage (0-100%)
- Higher is safer
- Color-coded badge

### Score Breakdown
Click on a route to see:
- **Crowd Density** - How crowded the route is
- **Infrastructure** - Road quality and development
- **Nearby Help** - Distance to emergency services
- **Lighting** - Street lighting and time of day

### Safety Badges
- **"Safest"** badge - Highest safety score among routes

## 🌃 Using 3D View

### Switching Views
1. Click "3D View" tab (top-left)
2. Wait for scene to load
3. Explore the city in 3D

### 3D Controls
- **Rotate** - Left click + drag
- **Pan** - Right click + drag (or two-finger drag on mobile)
- **Zoom** - Scroll wheel (or pinch on mobile)

### 3D Features
- **Buildings** - Real building shapes from OpenStreetMap
- **Roads** - Accurate road networks
- **Lights** - Street lights (visible at night)
- **People** - Moving pedestrians
- **Vehicles** - Traffic on roads

## 🚨 Emergency Features

### SOS Button
1. **Tap the red "SOS EMERGENCY" button** (always visible in sidebar)
2. A dialog will show:
   - Your current location
   - Nearest police stations
   - Emergency call button
3. Click "Call 100" to dial emergency services
4. Click "Cancel" to close

### Audio Safety Check
1. Click green "Safety Check" button
2. Listen to the voice prompt: "Are you safe?"
3. Click again to stop

## 💬 Using Chat Assistant

### Opening Chat
1. Click the blue chat bubble (bottom-right)
2. Chat window will slide up

### Quick Questions
Tap any quick question chip:
- "Best route to take?"
- "Is this area safe?"
- "Nearest police station?"
- "Show safe zones"

### Getting Answers
- Responses are instant
- Based on your current location/route
- Include actual safety scores
- Provide actionable advice

## ⚡ Performance Tips

### For Faster Loading
1. Use 2D map view for quick routing
2. Switch to 3D only when needed
3. Close chat when not in use
4. Avoid frequent location changes

### If Map is Slow
1. Zoom out slightly
2. Clear browser cache
3. Close other browser tabs
4. Refresh the page

## 🆘 Troubleshooting

### Map Not Loading
- Check internet connection
- Refresh the page
- Clear browser cache

### Search Not Working
- Type at least 2 characters
- Wait for auto-suggestions
- Try alternative name/spelling

### No Routes Found
- Check if locations are too close
- Verify locations are in coverage area
- Try nearby alternative locations

### 3D View Blank
- Wait for scene to load
- Check WebGL support in browser
- Try refreshing the page

### SOS Not Working
- Check browser permissions
- Ensure location services enabled
- Try manual location entry
